
# Wander In NKU —— 南开大学文化信息平台 🎓

**Wander In NKU** 是一款专为南开大学高考招生季设计的集成化信息导览平台。本项目基于 **C++** 和 **Qt 框架** 开发，旨在为考生、家长及访客提供一站式服务，包括招生政策查询、VR 校园全景漫游、校史文化深度体验以及专属的校园塔防小游戏。

> **课程名称：** 高级语言编程实训 2 (2024-2025 夏季学期)
> **所属学院：** 南开大学软件学院

## 🚩 项目背景
每年高招季，考生和家长往往面临信息获取分散、不对称以及无法实地考察校园的痛点。**Wander In NKU** 通过数字化手段，将南开大学官网、本科招生网、VR 全景平台等多源数据进行整合，以简洁直观的桌面应用形式，打破信息壁垒，传递“允公允能，日新月异”的南开精神。

## ✨ 核心功能

### 1. 信息导航与集成 🧭
- **一站式入口：** 无缝集成南开大学官网与本科招生网。
- **实时资讯：** 通过接口直接跳转官方数据源，确保招生政策、历年分数线等核心资讯的权威性与实时性。

### 2. 沉浸式 VR 校园体验 👓
- **云游南开：** 内置 VR 浏览模块，支持用户足不出户游览百年学府。
- **双版本支持：** 兼容旧版 720 云全景与新版官方全景平台，提供多角度的视觉体验。

### 3. 深厚文化底蕴 🏛️
- **校史时间轴：** 交互式展示南开百年发展历程。
- **南开精神：** 包含“公能精神”解读、“爱国三问”互动弹窗及“南开校训”展示。
- **名人堂：** 展示杰出校友生平，增强文化感召力。

### 4. 交互式 UI 设计
- **动态主页：** 首页采用自动轮播图展示校园风光（主楼、图书馆等）。
- **主题配色：** 整体 UI 采用南开校色（青莲紫）为主调，风格统一大气。

## 🎮 游戏模块：南开塔防
*（又名：领地之争 v1.0）*

为了增强平台的互动性和趣味性，我们开发了一款以“守护校园领地”为主题的策略类塔防游戏。

### 游戏玩法
- **目标：** 守护我方领地（蓝色方块），抵御敌方（红色方块/敌机）的侵袭。
- **资源系统：** 收集随机生成的 **阳光（南开校徽）** 来购买防御设施。
- **我方单位：**
  - **加农炮 1 (Cannon 1)：** 基础攻击单位。
  - **加农炮 3 (Cannon 3)：** 升级版单位，可向左前、正前、右前三个方向发射子弹。
  - **滚轮 (Roller)：** 防御型设施。
- **敌方单位：**
  - **红色方块：** 逐步侵蚀领地。
  - **敌机 (Enemy Plane)：** 飞越战场并投放新的敌人。
  - **云朵：** 战场环境障碍。
- **胜负判定：** 在规定时间内成功守住领地即为胜利，反之则失败。

### 技术实现亮点
- **自定义绘图：** 使用 `QPainter` 绘制网格、单位及动画效果。
- **对象池模式 (Object Pool)：** 在子弹管理中应用对象池技术，复用对象以减少内存开销，提升游戏流畅度。
- **碰撞检测：** 基于 `QRect::intersects` 实现子弹与敌方方块的精准碰撞逻辑。

## 🛠 技术栈与亮点

- **编程语言：** C++
- **开发框架：** Qt (Qt Widgets Application)
- **核心技术点：**
  - **界面管理：** 使用 `QStackedWidget` 实现多页面的无缝切换（栈式管理）。
  - **信号与槽 (Signals & Slots)：** 实现 UI 按钮与业务逻辑的解耦，处理跨模块通信。
  - **定时器 (QTimer)：** 驱动轮播图切换、游戏循环及敌军生成逻辑。
  - **绘图引擎 (QPainter)：** 游戏场景的实时渲染与动态更新。
  - **宏定义 (Macros)：** 在 `config.h` 中集中管理游戏参数（如刷新率、尺寸、路径），便于调试。

## 📂 项目结构

根据实验报告，核心代码结构如下：

```text
WanderInNKU/
├── main.cpp                # 程序入口
├── MainWindow/             # 主窗口框架
│   ├── mainwindow.h
│   └── mainwindow.cpp      # 负责 QStackedWidget 的页面调度
├── Pages/                  # 功能页面模块
│   ├── MainPage            # 首页（含轮播图逻辑）
│   ├── NankaiIntroPage     # 南开简介（校史、名人）
│   ├── AdmissionPage       # 招生信息（外部链接跳转）
│   ├── VRPageLite          # 校园 VR 浏览
│   └── CulturePage         # 文化专题（弹窗交互）
├── Game/                   # 塔防游戏引擎
│   ├── startscene          # 游戏开始菜单
│   ├── mainscene           # 游戏主场景（核心逻辑与绘图）
│   ├── config.h            # 游戏全局配置文件
│   ├── Items/              # 游戏对象类
│   │   ├── connon1/3       # 炮塔类
│   │   ├── bullet          # 子弹类
│   │   └── enemy_plane     # 敌机 AI 类
└── Resources/              # 资源文件（图片、图标、UI素材）
```

## 🚀 安装与构建

### 前置要求
- **Qt Creator** (推荐)
- **Qt 5.x 或 6.x** (搭配 MinGW 或 MSVC 编译器)

### 步骤
1.  克隆本项目代码：
    ```bash
    git clone <repository-url>
    ```
2.  使用 Qt Creator 打开项目文件（`WanderInNKU.pro` 或 `CMakeLists.txt`）。
3.  配置构建套件（Debug 或 Release）。
4.  点击 **运行 (Run)** 或按 `Ctrl+R` 进行编译和启动。
